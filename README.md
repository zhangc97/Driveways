# Driveways
### A web app that finds nearby Parking

## Distribution
* mobile work in progress (or until I learn to develop mobile)
* web is the web app

## Platform/Libraries
* React
* Firebase
* Google Maps API

## Usage
```
git pull https://github.com/zhangc97/Driveways.git
cd Web
npm install -r package.json
npm run start
```
